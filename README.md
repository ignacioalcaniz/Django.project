# QuantEdge

QuantEdge es una aplicación web desarrollada con Django orientada al análisis de activos financieros, inversión asistida por IA conceptual, registro de usuarios y dashboard privado.

## Funcionalidades

- Backend con Django.
- Base de datos MySQL/MariaDB.
- Registro, login y logout de usuarios.
- Dashboard privado protegido.
- Panel admin personalizado.
- Gestión de activos financieros.
- Compra simulada de acciones.
- Asesoramiento IA conceptual.
- Frontend maquetado con HTML, CSS3, Bootstrap y Sass.

## Tecnologías

- Python
- Django
- MySQL / MariaDB
- HTML
- CSS3
- Bootstrap
- Sass
- Pytest
